# Generate SCSS Project
`gen-project-scss`

### Installation

```sh
# (1)
curl

```

### Execution

**Create a new project**

```sh
gen-project-scss «your-project-name»
```

**Setup the watch-scss process**
(from `«/your-project-name»`)

```sh
watch-scss
```

- this will the `main.scss` file for changes
